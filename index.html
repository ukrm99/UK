Dova Protocol is an innovative endeavor aiming to unify liquidity between the Bitcoin network (BTC) and the EVM network.

Bitcoin EVM Bridge

The Dova Protocol integrates the MultiBit bridge, enabling users to seamlessly transfer BRC-20 tokens across chains via the MultiBit cross-chain bridge. Subsequently, users can leverage Dova Protocol on the Ethereum network to facilitate collateralized lending using the transferred BRC-20 tokens.

Dova loan Protocol

Security

Leveraging the security and reliability of the Bitcoin blockchain, the BRC-30S protocol ensures users' funds are secure, protected from malicious attacks or hacking threats.

Liquidity

The protocol enhances the liquidity of both Bitcoin and BRC-20 tokens, positioning itself to become one of the most popular and widely traded assets in the crypto market.

Scalability

The BRC-30S protocol enables users to access other blockchains such as Ethereum, Binance Smart Chain, Polygon, etc., providing more services and products like loans, games, NFTs, etc.

Passive Income

The protocol allows users to earn BRC-30S tokens as rewards for staking, potentially increasing the value of the tokens due to limited supply.
